# Hardware Specs App 

## პროექტის აღწერა
ეს არის Android აპლიკაცია, რომელიც მომხმარებელს აწვდის დეტალურ ინფორმაციას თანამედროვე პროცესორებისა (CPU) და ვიდეობარათების (GPU) შესახებ. აპლიკაცია იძლევა საშუალებას დავათვალიეროთ აპარატურის სპეციფიკაციები და მათი პერფორმანსის შეფასება.

## ტექნიკური დეტალები და არქიტექტურა
აპლიკაცია აგებულია თანამედროვე Android სტანდარტების დაცვით:
*   **UI Toolkit:** Jetpack Compose
*   **Architecture:** MVVM (Model-View-ViewModel) - ლოგიკა და UI სრულად განცალკევებულია.
*   **Local Storage:** Room Database ქეშირებისთვის და მონაცემების ლოკალურად შესანახად.
*   **Asynchronous Programming:** Kotlin Coroutines & Flow (StateFlow მონაცემების რეალურ დროში განახლებისთვის).
*   **Navigation:** Jetpack Compose Navigation სისტემა (ეკრანებს შორის არგუმენტების გადაცემით).

##  უნიკალური ფუნქციონალი (New Feature)
პროექტში ინტეგრირებულია **Compose Canvas API**, რომლის საშუალებითაც `HardwareDetailScreen`-ზე იხატება დინამიური **"Performance Gauge"** (პერფორმანსის მაჩვენებელი რკალი). ის ავტომატურად იცვლის ფერსა და შევსების დონეს მოცემული ტექნიკის წარმადობის ქულის (Performance Score) მიხედვით, რაც ვიზუალურად ბევრად უფრო ეფექტურს ხდის მონაცემთა პრეზენტაციას ვიდრე სტანდარტული ტექსტი.
